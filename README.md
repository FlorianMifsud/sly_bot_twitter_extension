## Installation

```bash
  git clone https://github.com/FlorianMifsud/sly_bot_twitter_extension.git
```

```bash
  cd sly_bot_twitter_extension
```

```bash
  pnpm i
```

```bash
  pnpm run start
```

## Running Tests

To run tests, run the following command

```bash
  pnpm run test
```
